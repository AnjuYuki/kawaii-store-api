## Fetching Product Data with fetch

Let's fetch a single product to see how the API works.

```js title="JavaScript" lineNumbers
const response = await fetch(
  'https://kawaii-store-api.vercel.app/api/products/1',
);
const product = await response.json();
console.log(product);
```

Copy the code and run it in your browser's console.

### Response

```json
{
  "id": 1,
  "title": "くまさんメロンパン",
  "description": "くまの形をした、ふんわり甘いメロンパン。",
  "price": 260,
  "currency": "JPY",
  "category": "bread",
  "images": ["/images/categories/bread.png"],
  "isNew": true,
  "isPopular": true,
  "rating": { "rate": 4.7, "count": 132 }
}
```

This returns the product data for `id` `1`.
